# RouteMapperWeb

## Wymagania:
- Node.js
- Angular

## Uruchamianie:
### Serwer
- ``npm server``
Serwer powinien uruchomić się na porcie 8080
### Frontend
- ``npm i``
- ``npm start``
Frontend powinien uruchomić się na porcie 4200
